Nokia N900折腾笔记[1]-刷入系统

一直想买一个Linux的掌上全键盘终端而且资金有限. 某宝上正好有Nokia N900, 了解其系统Maemo是基于Debian的之后, 我动心了… 遂剁手得之.

但卖家很明显懒得刷机(Maemo没有格机这一个功能), 所以机器到手后里面全是前使用者的个人信息, 银行卡号, 姓名, 电话号, 家里的车型号, 前用户是广东人, 今年38岁等等等等. 很乱, 于是得重新刷机……

碍于机器停产和Nokia的消极政策, 目前官方的源已经死掉了, 还有官方的维护地址, 于是去Dospy爬贴. 发现DNE是个好东西. 但是其2010的EMMC已经失效, 于是我再次爬贴, 发现一篇帖子的回复里连接了一个有效的刷机包的链接, 遂下载.

差点被坑. 一般来说都是先刷EMMC后刷ROM, 但是那个包里面的1.bin是ROM, 2.bin是EMMC………打开一看便知道.
于是, 开始刷机.

  • 刷机环境: Thinkpad X61 Windows XP SP3

    (本文所提到的任何资源均重新打包并上传到百度云和MEGA)
    话说原本我的X61上是Arch…为了折腾N900特地安装了XP.

1. 首先, 安装libusb-win32, 这是Flasher必备的依赖.

2. 然后, 准备好rom和emmc并重命名方便操作.

3. 安装Flasher.

4. 将Flasher的安装目录加入环境变量并测试Flasher-3.5.exe能否在cmd下运行.

5. 将N900关机, 按住U键并插入数据线, 直到N900屏幕右上角显示一个很小的USB图标, 松开U. 此时电脑端应该反复右下角刷弹窗, 等他搜完.


6. 输入指令:flasher-3.5 -F “emmc目录” -f -R


7. 静候自动重启

8. 然后重复第五个步骤, 进入Update模式并参考第六个步骤刷入ROM文件. 成功之后如图:

9. 重启后便大功告成啦!!
